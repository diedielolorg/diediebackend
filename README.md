# DieDie-Backend

## 프로젝트 소개

- 롤을하면서 한번이라도 누군가와 싸워본적 있으신가요?
- 이름도 모르는 사람에게 욕을 먹고 기분이 상해 고소장을 작성해본적 있으신가요?
- DieDie는 익명 뒤에 숨어서 남에게 욕을하는 유저들을 신고하여 박제 할 수 있는 서비스입니다!

### 주요기능

- 욕설 랭킹
- 인게임 정보
- 욕설 신고

  </br>
